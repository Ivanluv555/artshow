=====================================
功能模块映射
=====================================

1. 用户认证流程:
   login_screen.dart → auth_service.dart → api_service.dart → 后端 /user/login

2. 主页展示:
   home_screen.dart → api_service.dart → 后端 /product/list + /artcate/list
   └── product_card.dart (产品卡片)
   └── category_chip.dart (分类标签)

3. 课程模块:
   courses_screen.dart → api_service.dart → 后端 /course/list
   └── course_card.dart
   course_detail_screen.dart → api_service.dart → 后端 /course?courseId=X

4. 讲师模块:
   instructors_screen.dart → api_service.dart → 后端 /instructor/list
   └── instructor_card.dart
   instructor_detail_screen.dart → api_service.dart → 后端 /instructor?instructorId=X

=====================================
数据流向
=====================================

用户操作 → Screen (页面) → Service (服务层) → API → 后端
                ↓
            Widget (组件) ← Model (数据模型) ← API 响应

=====================================
状态管理
=====================================

Provider (全局)
└── AuthService
    ├── isLoggedIn (登录状态)
    ├── token (JWT Token)
    └── login/logout 方法

=====================================
API 配置
=====================================

位置: lib/utils/constants.dart

ApiConstants.baseUrl:
  • Android 模拟器: http://10.0.2.2:8888
  • iOS 模拟器: http://localhost:8888
  • 真实设备: http://你的IP:8888
